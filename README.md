## 👏说明

本仓库是对 STEAM 爆款游戏 [Helltaker](https://store.steampowered.com/app/1289310/Helltaker/) 中文汉化项目，主要负责为大家提供免费优质的中文汉化包。

如果大家对本项目中任何地方有疑问或者建议，都可以在 [Issues](https://github.com/SeaEpoch/Helltaker-Chinese/issues) 中提出你的想法。

## ✅如何安装？

这非常的简单，你只需要在 Steam 中找到游戏，然后浏览游戏根目录，最后将项目中的 local 文件夹与 localHM 文件夹替换游戏根目录中的同名文件夹即可。
![在 Steam 中打开游戏根目录](/docs/SeaEpoch@2025-Nov-13-220613734.jpg)
![替换文件](/docs/SeaEpoch@2025-Nov-13-220921354.jpg)
